# ToDo-App-Favorite
Simple # ToDo-App-Favorite

Simple interactive To-Do List app built with HTML, CSS, and JavaScript.  
Features: add, complete, delete tasks; localStorage persistence; alerts when all tasks are done.

## How to run
1. Clone the repo
2. Open index.html in your browser
